# Blood Donation Management System

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Features
- Donor registration
- Blood search
- Blood drive promotion
- Testimonials
- Urgent blood requests

## How to Run
1. Install XAMPP
2. Move project folder to htdocs
3. Import database.sql into phpMyAdmin
4. Start Apache and MySQL
5. Open localhost/projectname
